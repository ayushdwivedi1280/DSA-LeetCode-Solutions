# Greedy Algorithms

This folder contains problems based on the **Greedy Algorithm** approach.

In this category, we solve questions by making the **best choice at each step** to achieve an optimal solution.

Problems included here will help in understanding:

* Local optimal → Global optimal decisions
* Efficient problem-solving techniques
* Common greedy patterns used in coding interviews

---

**Note:** Each problem in this folder belongs to the Greedy category and focuses on building strong intuition for greedy strategies.
